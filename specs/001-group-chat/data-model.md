# Data Model: Group Chats

**Feature**: `001-group-chat`  
**Date**: 2026-05-04

---

## Firestore Collections

### `groups` (top-level collection)

**Path**: `groups/{groupId}`

| Field | Type | Nullable | Notes |
|-------|------|----------|-------|
| `id` | String | No | Same as document ID |
| `name` | String | No | Group display name; duplicates allowed |
| `imageUrl` | String | No | Empty string when no image |
| `members` | List\<String\> | No | UIDs of all members; used for arrayContains query |
| `membersEmails` | List\<String\> | No | Emails of members; stored as-is, no validation |
| `admins` | List\<String\> | No | UIDs of admins; creator is always first |
| `lastMessage` | String | Yes | Preview text; empty string on creation |
| `lastMessageTime` | Timestamp | Yes | null on creation; updated on each send |
| `createdAt` | Timestamp | No | Set at creation time |

**Firestore index required**: `members` array-contains queries are served by default Firestore single-field indexes. No composite index needed for basic usage.

---

### `groups/{groupId}/messages` (subcollection)

**Path**: `groups/{groupId}/messages/{messageId}`

| Field | Type | Nullable | Notes |
|-------|------|----------|-------|
| `id` | String | No | Same as document ID |
| `senderId` | String | No | UID of the sending user |
| `senderEmail` | String | No | Email of the sending user |
| `text` | String | No | Message body; non-empty enforced in UI |
| `createdAt` | Timestamp | No | Set at send time |

**Ordering**: Client queries use `orderBy('createdAt', descending: false)` â€” oldest messages first (top to bottom in the chat view).

**Firestore index required**: `createdAt` ascending on the messages subcollection. Firestore auto-creates single-field indexes; no manual composite index needed unless additional filters are added later.

## Dart Model Classes

### `GroupModel`

```
class GroupModel {
  final String id;
  final String name;
  final String imageUrl;
  final List<String> members;
  final List<String> membersEmails;
  final List<String> admins;
  final String? lastMessage;
  final DateTime? lastMessageTime;   // @JsonKey with Timestamp converter
  final DateTime? createdAt;          // @JsonKey with Timestamp converter
}
```

- `fromJson` / `toJson` generated by `json_serializable`
- `fromFirestore({required String id, required Map<String,dynamic> data})` merges id into the map before `fromJson`

## Firestore Path Constants

```dart
// lib/constants/fierstore_paths.dart  (additions)
const String groupsCollection = 'groups';
const String messagesCollection = 'messages';
```

Path formulas used in the data source:
- Group list: `groupsCollection`
- Single group: `'$groupsCollection/$groupId'`
- Message list: `'$groupsCollection/$groupId/$messagesCollection'`
- Single message: `'$groupsCollection/$groupId/$messagesCollection/$messageId'`
